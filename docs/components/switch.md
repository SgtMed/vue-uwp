## Basic and Disabled

<ex-switch></ex-switch>
```html
  <vup-switch v-model="val" /> Switch
  <vup-switch disabled v-model="val2" /> Disabled Switch
```

## Configuration

### `v-model`
Default: `false`

The variable indicating the state of the switch.

### `disabled`
Default: `false`

Determines whether the button should be in disabled mode or not.
